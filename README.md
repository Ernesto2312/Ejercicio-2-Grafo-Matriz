# Ejercicio 2 – Estructura de Datos - Grafo con Matriz de Adyacencia.

## 📌 Descripción
Este ejercicio implementa una estructura de datos correspondiente al tema 2 de la asignatura.  
Incluye operaciones fundamentales, manejo de datos y pruebas de funcionamiento.

<img width="541" height="627" alt="image" src="https://github.com/user-attachments/assets/148436dd-7a80-42ba-a602-53e67f78b3a0" />

Programa en Java que modela un grafo no dirigido utilizando una
matriz de adyacencia y ejecuta los recorridos BFS (Búsqueda en Anchura) y DFS
(Búsqueda en Profundidad).

## 🧩 Funcionalidades
- Crear la estructura de datos
- Insertar elementos
- Eliminar elementos
- Buscar o recorrer la estructura
- Mostrar el estado actual
- Manejo de casos especiales (si aplica)

## 🛠️ Tecnologías
- Lenguaje: **Java**
- Paradigma: Programación orientada a objetos


## ▶️ Ejecución
Compilar:
javac src/*.java

java src/Main
